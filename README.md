# ScholarZone (Backend)

ScholarZone is a Scholarship Application and Management System designed to simplify the process of finding, applying for, and reviewing scholarships. This repository contains the reviewer-side application.

## Features
* User authentication (login & signup)
* Create scholarship applications
* Review applicants
* Approve/reject reviewers (admin)
* Approve/reject scholarship applications (admin)
* Profile management
* AI analysis results display

## Tech Stack
Language: JavaScript\
Runtime: Node.js\
Framework: Express.js\
Architecture: MVC
